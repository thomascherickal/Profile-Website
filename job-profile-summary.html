The Transition

For six years, I was a writer. I published 500+ articles across HackerNoon, Medium, Substack, Hashnode, and my canonical home at thomascherickal.com. I explained Generative AI, Quantum Computing, and emerging technologies to audiences who needed clarity.

But something was missing. You can write about systems. You can analyze architecture. You can critique trade-offs. But until you sit down and actually build, you're describing terrain you've never walked.

So I transitioned. And here's what that looks like: I bring the clarity of a writer into production code. Rigorous thinking. Zero tolerance for hand-waving. The ability to name what things are. And the understanding that the best code is code your team can reason about, scale, and extend without me having to maintain it forever.

What I Build

SLMs & LLMs

Small Language Models and Large Language Models are the backbone of the AI stack right now. Both need infrastructure.

  • In Python: RAG pipelines that retrieve the right context, agentic frameworks using OpenFang and MCP protocols, fine-tuning systems for knowledge distillation, and local model deployment stacks.
  • In Rust: High-performance token streaming engines for real-time responses, inference schedulers that handle concurrent requests without melting, embedding pipelines, and quantization tooling.
AI Ecosystem Infra

The entire infrastructure that makes models useful. Python handles the logic; Rust handles the speed.

  • In Python: Request routing across model providers, response caching, token accounting, monitoring, observability, and vector DB integrations.
  • In Rust: The performance layer, low-latency inference orchestration, batch processing pipelines, memory-efficient loading, and concurrent request handling.

The Discipline

I build with Test-Driven Development (TDD) as default. 100% code coverage is the target. Not because numbers look good in dashboards—they don't. But because every line represents a decision about correctness. If I haven't tested it, I don't understand it.

The code ships with docstrings, type hints, architecture decisions recorded, SOLID principles, and modular design. Someone will read this six months from now; they should understand it without me.

AI-Assisted Development

I work alongside Claude Code, GitHub Copilot, Google Antigravity, and emerging IDE tools. Not as a crutch, but as a thought partner. These tools surface patterns, catch boilerplate, and let me spend less time on scaffolding and more time on the hard problems: correctness, performance, and maintainability. But the design, testing, and ultimate responsibility are entirely mine.

What I'm Building

Note: All repositories listed below are upcoming open-source releases.

Release schedule and roadmap for the Generative AI and SLM Stack:

Q3 2026

slm-inference-engine Rust

A highly optimized, CPU-accelerated local SLM inference framework written in pure Rust. Built to stream context-aware token completions under 15ms on 2–8 GB low-spec consumer RAM targets, bypassing heavy Python runtime dependencies completely.

ai-career-toolkit Python

An automation suite orchestrating Python agents and advanced prompting workflows to operationalize RECRUITED. Features full-pipeline resume optimization via structured schema generation, automated LinkedIn profile audits, and NotebookLM-compatible knowledge-base generators.

quantum-ai-experiments Python

A collection of production-grade research notebooks integrating classical machine learning with quantum algorithms. Explores variational classifiers on Qiskit, hybrid pipeline scheduling on the Quantinuum stack, and cryptographic threat simulation against blockchain consensus.

digital-futurist-site HTML/CSS

A high-performance, responsive portfolio hub featuring strict JSON-LD semantic structure and automated SEO/GEO optimization pipelines. Features custom glassmorphic styling, responsive layout grids, and embedded cross-platform social syndication tags.

rust-llm-router Rust

An ultra-low-latency API gateway in Rust for high-throughput LLM routing. Implements active health-checks, rate-limit throttling, automatic fallback failovers, and cost-minimization algorithms to route requests dynamically across multiple API providers.

Q4 2026

vector-forge Rust

A blazing-fast, dependency-free vector mathematics library implementing SIMD intrinsics in Rust. Built specifically to handle dense embeddings, cosine similarity computations, and approximate nearest neighbor searches via custom high-speed HNSW indexing implementations.

ai-cli Rust

A terminal-native developer workspace copilot built in Rust. Features interactive token streaming, sandboxed CLI execution, system-level tool calling via local llama.cpp bindings, and an extensible WASM-based plugin architecture for custom automation commands.

neural-bench Python

A diagnostic suite testing local LLM performance across edge devices. Automatically detects host hardware profiles, stress-tests memory bandwidth, and measures time-to-first-token (TTFT), tokens-per-second throughput, and factual generation quality metrics.

agent-memory-kit Python

A production memory middleware for autonomous agents utilizing LanceDB and Nomic embeddings. Implements vector-based episodic recall, hierarchical semantic summarization, and time-decay relevance filters pluggable directly into LangChain, LlamaIndex, or raw APIs.

slm-distiller Python

A data-distillation pipeline for knowledge transfer from frontier models to compact local SLMs. Automates synthetic dataset generation, logit matching, and LoRA/QLoRA training scripts to run on consumer hardware with under 8GB VRAM.
